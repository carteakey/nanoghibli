# NanoGhibli Requirements

## 1. Frame Extraction & Input
- **Videos**: Capture scale stills of every moving frame (whenever the frame changes).
- **Photos**: Take in photos from a folder.

## 2. Stylization
- Pass the captured frames/photos to the Gemini API ("nano banana two") to get Ghibli-type images.

## 3. Animation & Assembly
- Animate the stylized images based on reference material (like the original reference video).
- Create a Ghibli-style trailer.

## Tone & Intent
- This is an homage to Studio Ghibli, not a copy.
- Not for monetization; strictly for visualizing how things would look if Ghibli created them (without using their actual art/touch).
